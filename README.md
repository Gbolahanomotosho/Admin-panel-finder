# Admin panel finder 


  This is an admin panel finder with an inbuilt wordlist of over 1600+


  It can find the admin panel of a website


  I wont be responsible for the illegal use of this tool



# Command


 git clone https://github.com/Gbolahanomostosho/Admin-panel-finder




 cd Admin-panel-finder



 pip install -r requirements.txt




 python Admin-panel-finder.py
